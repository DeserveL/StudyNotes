## 装饰模式

**定义**：在不必改变原类文件和使用继承的情况下，动态地扩展一个对象的功能。它是通过创建一个包装对象，也就是装饰来包裹真实的对象

1. 不改变原类文件。
2. 不使用继承。
3. 动态扩展。
